# nnc - nekobato no Web Components

A collection of Web Components by @nekobato

## Components

- `nnc-e-display`: A Lit-based Japanese train-sign inspired marquee display.

## Development

```sh
npm install
npm run docs:dev
```

## Quality Checks

```sh
npm test
npm run lint
npm run typecheck
npm run format
npm run build
npm run docs:build
```

## GitHub Pages

The VitePress site is configured with `base: "/nn-web-components/"`.
GitHub Pages deployment is handled by `.github/workflows/deploy-pages.yml`.
